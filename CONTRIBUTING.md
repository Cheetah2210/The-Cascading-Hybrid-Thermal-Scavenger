# Contributing to CHTS

Thank you for helping optimize the Cascading Hybrid Thermal Scavenger! 

## Ways to Contribute

1. **Fluid Dynamics Validation:** Help refine the expansion math or suggest optimal non-toxic conductive fluid suspensions.
2. **CAD Refinements:** Submit physical design optimizations for the 3D-printable channel blocks to improve flush electrode mounting.
3. **Simulation Improvements:** Add more granular thermodynamic variables to the Python modeling scripts.

## Contribution Workflow

* Fork the repository.
* Create a new branch for your feature or fix.
* Keep your code clean and document any hardware dimension changes.
* Submit a Pull Request describing your engineering updates.

All hardware contributions must respect the CERN Open Hardware Licence v1.2 terms.
